## 📌 Sobre o Projeto

Projeto desenvolvido para a entrega de um trabalho acadêmico, consistindo em um site interativo de quizes.

## 🛠 Tecnologias utilizadas:

O site foi construído com foco em interatividade e organização. As tecnologias utilizadas foram:

## 📚 Alguns conceitos aplicados

Neste projeto apliquei os seguintes pontos:

✔️ Manipulação do DOM com JavaScript;

✔️ Lógica de programação para validação de respostas;

✔️ Estruturação semântica com HTML;

✔️ Estilização com CSS;

✔️ Organização e clareza na experiência do usuário.
